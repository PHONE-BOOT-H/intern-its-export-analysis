# Method First — 방법론 우선 조사

"너는 전문가야"라고 역할만 주는 것이 아니라,
실제 방법론과 평가 기준을 먼저 조사한 뒤 그 기준으로 작업합니다.

## 실행 단계

### Step 1: 도메인 파악
작업이 속한 분야를 정의:
- 이것은 어떤 종류의 문제인가?
- 관련 분야의 표준 접근법은 무엇인가?
- 어떤 평가 기준이 있는가?

### Step 2: 방법론 조사
웹에서 다음을 조사:
- 업계 표준 방법론 (있는 경우)
- 관련 프레임워크나 패턴
- 주요 성공/실패 케이스
- 측정 가능한 평가 기준

### Step 3: 조사 결과 정리
```markdown
## 방법론 조사: [주제]

### 업계 표준 접근법
- 방법 A: [설명, 장단점]
- 방법 B: [설명, 장단점]

### 평가 기준
- 기준 1: [설명]
- 기준 2: [설명]

### 추천 접근법
[이유와 함께]

### 주의사항
[알려진 함정, 실패 패턴]
```

### Step 4: 사용자 확인
AskUserQuestion:
- 조사한 내용 설명
- 추천 접근법 제시
- 사용자의 컨텍스트에 맞게 조정 필요한지 확인

### Step 5: 기준 기반 실행
합의된 방법론과 평가 기준을 적용해서 작업 진행.
작업 중 기준에서 벗어나는 경우 즉시 알림.
